💡 Le prompt parfait — images ultra-réalistes
Osiris Vision est réglé pour dépasser le rendu des IA classiques — mais 90 % du résultat vient de votre description. La recette pro :

1️⃣ Le sujet, précis — qui/quoi, âge, matière, couleur, action.
❌ "un chien" → ✅ "un berger australien aux yeux vairons, courant dans l'herbe haute"

2️⃣ Le style — photo réaliste, cinématique, portrait studio, macro, drone, peinture à l'huile, 3D Pixar…

3️⃣ La lumière — golden hour, contre-jour, néons bleus, lumière douce de fenêtre, éclairage studio… C'est LE secret du réalisme.

4️⃣ Le cadrage — gros plan, plan large, vue aérienne, 50mm, faible profondeur de champ (bokeh)…

5️⃣ Les mots-clés qualité — terminez par : "ultra détaillé, photoréaliste, 8K, netteté parfaite, qualité professionnelle".

🏆 Exemple complet à copier :
"Portrait d'une femme rousse de 30 ans, taches de rousseur, regard vers l'objectif, photo réaliste style studio, lumière douce latérale, fond gris neutre, objectif 85mm, bokeh, ultra détaillé, photoréaliste, 8K"

⚙️ Choisissez aussi le format (Carré / Paysage / Portrait) au-dessus du champ avant d'envoyer.
